#vim
# 一、Vim简介
- **Vim** 是一个历史悠久的**模态文本编辑器**，它使用不同的模式（如普通模式、插入模式）来区分命令和文本输入，以实现极致高效的键盘操作。
- 而 **Obsidian 的 Vim 模式**则**不是**一个独立的 Vim 编辑器，而是让 Obsidian 的内置编辑器**采用 Vim 的键绑定和操作逻辑**。
~~- 其主要作用是让熟悉 Vim 的用户能在 Obsidian 中沿用高效的键盘操作习惯，或者为追求纯键盘操作和快速文本编辑的用户提供一套强大的工具。~~
- 使用 Vim 模式可以让你通过键盘命令更高效地在文本中导航和编辑，减少对鼠标的依赖。虽然学习曲线相对陡峭，但一旦掌握了基础，会极大地提升你的文本处理速度。

# 二、开启Vim模式
1. 打开 Obsidian。 
2. 点击左下角的 **设置 (Settings)** 图标。
3. 在设置菜单中，选择左侧的 **编辑器 (Editor)**。
4. 向下滚动，找到 **Vim 键绑定 (Vim key bindings)** 的开关。
5. 输入测试题答案":q!"  打开这个开关。

# 三、基础操作命令
##  1、Vim 的核心概念 - 模式 (Modes)**

Vim 最重要的概念就是“模式”。你在不同的模式下，键盘按键的功能是不同的。在 Obsidian 的 Vim 模式中，你主要会用到以下几种模式：

1. **普通模式 (Normal Mode):** 这是你开启 Vim 模式后的默认模式。在这个模式下，你输入的按键**不是**直接输入文本，而是执行各种命令，用于导航、删除、复制、粘贴等。
2. **插入模式 (Insert Mode):** 在这个模式下，你输入的按键会像普通文本编辑器一样直接插入文本。
3. **可视模式 (Visual Mode):** 用于选择文本。进入可视模式后，你的移动会高亮选择文本，然后你可以对选择的文本执行删除、复制等操作。
	    - `o`: 在当前行下方插入一个新行并进入插入模式。
	    - `o`: 在当前行下方插入一个新行并进入插入模式。
## 2、如何在模式之间切换**
- 从 **普通模式** 进入 **插入模式**:
    - `i`: 在当前光标前插入。
    - `a`: 在当前光标后插入。
    - `o`: 在当前行下方插入一个新行并进入插入模式。
    - `O`: 在当前行上方插入一个新行并进入插入模式。
    - `I`: 在当前行的第一个非空白字符前插入。
    - `A`: 在当前行的末尾插入。
- 从 **插入模式** 或 **可视模式** 进入 **普通模式**:
    - `Esc` 键。 (或者按 `Ctrl+[`)
- 从 **普通模式** 进入 **可视模式**:
    - `v`: 进入字符可视模式，按键移动会按字符选择。
    - `V`: 进入行可视模式，按键移动会按行选择。
    - `Ctrl+v`: 进入块可视模式（在 Obsidian 中可能支持有限）。
