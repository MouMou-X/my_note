### 1. 对象 (Object)

在 JavaScript 中，**对象**是**键值对的集合**。它是一种复合数据类型，用于存储多个相关数据和功能。你可以把它想象成一个“盒子”，里面装了各种标签（键）和对应的值（数据或功能）。

- **特点：**
    - 无序的属性集合。
    - 可以存储各种数据类型（字符串、数字、布尔值、数组、甚至其他对象）。
    - 可以包含函数（被称为方法）。
- **字面量表示法：** 使用花括号 `{}`。

**示例：**

JavaScript 

```
let person = {
  name: "张三",       // 键是 name，值是 "张三"
  age: 30,          // 键是 age，值是 30
  isStudent: false, // 键是 isStudent，值是 false
  hobbies: ["阅读", "跑步"], // 键是 hobbies，值是一个数组
  greet: function() { // 键是 greet，值是一个函数（方法）
    console.log("你好，我是 " + this.name);
  }
};
```

在这个例子中，`person` 就是一个对象。

### 2. 属性 (Property)

**属性**是对象中存储的数据（值）。每个属性都有一个**键**（也称为属性名）和一个**值**。你可以通过属性名来访问这些值。

- **访问方式：**
    - 点表示法 (`.`)：`object.propertyName`
    - 方括号表示法 (`[]`)：`object['propertyName']`

**示例（基于上面的 `person` 对象）：**

- `name` 是 `person` 对象的一个属性，它的值是 `"张三"`。
- `age` 是 `person` 对象的一个属性，它的值是 `30`。
- `hobbies` 是 `person` 对象的一个属性，它的值是 `["阅读", "跑步"]`。

**访问属性：**

JavaScript

```
console.log(person.name);       // 输出: 张三
console.log(person["age"]);     // 输出: 30
```

### 3. 函数 (Function)

**函数**是一段可重复使用的代码块，它执行特定的任务。函数可以接受输入（称为参数），并可以返回一个结果。

- **特点：**
    - 封装代码，提高代码复用性。
    - 可以接受零个或多个参数。
    - 可以返回一个值（如果没有明确返回，则返回 `undefined`）。
- **定义方式：**
    - 函数声明：`function functionName(parameters) { /* code */ }`
    - 函数表达式：`let functionName = function(parameters) { /* code */ }`
    - 箭头函数 (ES6+)：`let functionName = (parameters) => { /* code */ }`

**示例：**

JavaScript

```
// 函数声明
function add(a, b) {
  return a + b;
}

// 函数表达式
let multiply = function(x, y) {
  return x * y;
};

// 箭头函数
let subtract = (a, b) => a - b;

console.log(add(5, 3));        // 输出: 8
console.log(multiply(4, 2));   // 输出: 8
console.log(subtract(10, 3));  // 输出: 7
```

### 4. 方法 (Method)

**方法**是**作为对象属性的函数**。当一个函数被存储为对象的一个属性时，我们就称它为这个对象的方法。方法通常用于对对象内部的数据执行操作，或者与对象相关的功能。

- **特点：**
    - 它是一个函数，但它依附于一个对象。
    - 它可以通过 `this` 关键字访问它所属对象的其他属性和方法。
- **调用方式：** 使用点表示法 (`.`)，后面跟着括号 `()`。

**示例（基于上面的 `person` 对象）：**

- `greet` 是 `person` 对象的一个**方法**。

**调用方法：**

JavaScript

```
person.greet(); // 输出: 你好，我是 张三 (这里的 `this.name` 指向 `person.name`)
```

### 总结关系：

- **对象**是“盒子”。
- **属性**是“盒子里的东西” (数据)。
- **函数**是“可以独立执行的动作”。
- **方法**是“盒子里的可以执行的动作”，并且这个动作通常与这个盒子本身相关联。

在 Templater 的 `tp` 对象中，这些概念表现得淋漓尽致：

- `tp` 是一个**对象**。
- `tp.file` 是 `tp` 对象的一个**属性**（它本身又是一个对象）。
- `tp.file.title` 是 `tp.file` 对象的一个**属性**（它是一个字符串值，代表文件标题）。
- `tp.date.now()` 中的 `now` 是 `tp.date` 对象的一个**方法**（它是一个函数，执行后返回当前日期）。
