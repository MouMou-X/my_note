# 如何在模式之间切换

- 从 **普通模式** 进入 **插入模式**:
    - `i`: 在当前光标前插入。
    - `a`: 在当前光标后插入。
    - `o`: 在当前行下方插入一个新行并进入插入模式。
    - `O`: 在当前行上方插入一个新行并进入插入模式。
    - `I`: 在当前行的第一个非空白字符前插入。
    - `A`: 在当前行的末尾插入。
- 从 **插入模式** 或 **可视模式** 进入 **普通模式**:
    - `Esc` 键。 (或者按 `Ctrl+[`)
- 从 **普通模式** 进入 **可视模式**:
    - `v`: 进入字符可视模式，按键移动会按字符选择。
    - `V`: 进入行可视模式，按键移动会按行选择。
    - `Ctrl+v`: 进入块可视模式（在 Obsidian 中可能支持有限）。

# 基础的 Vim 命令 (在 普通模式 下执行)

记住，以下命令大多数是在 **普通模式** 下使用的。

**1. 光标移动:**

- `h`: 向左移动一个字符。
- `j`: 向下移动一行。
- `k`: 向上移动一行。
- `l`: 向右移动一个字符。
- `w`: 移动到下一个单词的开头。
- `b`: 移动到上一个单词的开头。
- `e`: 移动到下一个单词的结尾。
- `0`: 移动到当前行的开头。
- `^`: 移动到当前行的第一个非空白字符。
- `$`: 移动到当前行的末尾。
- `gg`: 移动到文件的第一行。
- `G`: 移动到文件的最后一行。
- `行号G`: 移动到指定行号（例如，`10G` 移动到第10行）。
- `Ctrl+f`: 向下翻页。
- `Ctrl+b`: 向上翻页。
- `{`: 移动到上一个段落/代码块的开头。
- `}`: 移动到下一个段落/代码块的开头。

**2. 删除、复制、粘贴 (操作符 + 移动命令 或 文本对象):**

Vim 的强大之处在于可以将“操作符”与“移动命令”或“文本对象”结合使用。

- **操作符:**
    
    - `d`: 删除 (delete)
    - `c`: 修改 (change) - 删除并进入插入模式
    - `y`: 复制 (yank)
    - `p`: 在光标**后**粘贴 (paste)
    - `P`: 在光标**前**粘贴
- **结合移动命令:**
    
    - `dw`: 删除到下一个单词开头。
    - `de`: 删除到下一个单词结尾。
    - `d$`: 删除到行尾。
    - `d0`: 删除到行首。
    - `dd`: 删除当前整行。
    - `yw`: 复制到下一个单词开头。
    - `yy`: 复制当前整行。
    - `y$`: 复制到行尾。
    - `yw`, `yy` 等命令会将内容复制到剪贴板，然后可以使用 `p` 或 `P` 粘贴。
- **结合文本对象 (非常有用的高级技巧):**
    
    - `diw`: 删除当前光标所在的**内部单词** (delete inner word)。
    - `daw`: 删除当前光标所在的**整个单词** (delete a word，包含后面的空格)。
    - `di"`: 删除双引号内部的内容 (delete inner double quotes)。
    - `da'`: 删除单引号内部的内容 (delete a single quotes，包含引号本身)。
    - `di(` 或 `di)`: 删除括号内部的内容。
    - `da(` 或 `da)`: 删除括号及括号内部的内容。
    - `di[` 或 `di]`: 删除方括号内部的内容。
    - `da[` 或 `da]`: 删除方括号及方括号内部的内容。
    - `di{` 或 `di}`: 删除花括号内部的内容。
    - `da{` 或 `da}`: 删除花括号及花括号内部的内容。
    - `dip`: 删除当前光标所在的**内部段落** (delete inner paragraph)。
    - `dap`: 删除当前光标所在的**整个段落** (delete a paragraph)。
    - `ciw`, `caw`, `ci"`, `ca'`, `ci(`, `ca)`, `cip`, `cap` 等：作用类似 `d`，但删除后会立即进入**插入模式**。
    - `yiw`, `yaw`, `yi"`, `ya'`, `yi(`, `ya)`, `yip`, `yap` 等：作用类似 `y`，复制相应的文本对象。

**3. 其他常用命令:**

- `u`: 撤销 (undo)。
- `Ctrl+r`: 重做 (redo)。
- `.`: 重复上一个修改命令。例如，如果你刚刚使用了 `dd` 删除一行，再按 `.` 会再删除下一行。
- `:`: 进入命令行模式（在 Obsidian 中可能支持有限的命令）。

**5. 使用可视模式 (Visual Mode):**

1. 在普通模式下，按 `v` 进入字符可视模式。
2. 使用移动命令 (`hjkl`, `wbe`, `0^$`, etc.) 来选择文本。
3. 选择完成后，可以按一个操作符，例如 `d` (删除选择的内容)，`y` (复制选择的内容)。
4. 按 `Esc` 退出可视模式回到普通模式。

**练习技巧:**

- **从基础开始:** 不要试图一次记住所有命令。先专注于最基本的移动 (`hjkl`) 和模式切换 (`i`, `Esc`)，以及简单的删除 (`dd`, `dw`) 和复制粘贴 (`yy`, `p`)。
- **多练习:** 每天花一些时间在 Obsidian 中使用 Vim 模式来写笔记。
- **使用 Vim Tutor (可选):** 如果你想系统地学习 Vim，可以尝试安装一个真正的 Vim 编辑器，然后运行 `vimtutor` 命令。这是一个交互式的教程，非常适合初学者。
- **查阅速查表:** 网上有很多 Vim 命令的速查表，可以作为参考。
