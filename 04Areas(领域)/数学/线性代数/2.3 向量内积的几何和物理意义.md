# 向量内积

## 几何解释
- 向量a在向量b方向上的投影与向量b的模的乘积。
- 公式表示为：$a \cdot b = |a||b|\cos\theta$

## 代数表示
- 以三维向量为例，$\vec{a} = (a_x, a_y, a_z)$，$\vec{b} = (b_x, b_y, b_z)$
- 公式表示为：$a \cdot b = a_x b_x + a_y b_y + a_z b_z$

### 特殊情况：求向量a的长度
- 向量a与其自身的内积等于其模的平方。
- $$|\vec{a}| = \sqrt{\vec{a} \cdot \vec{a}} = \sqrt{a_x^2 + a_y^2 + a_z^2}$$
## 向量内积的物理解释
- **采购**: 以一周买菜为例，`价格向量`与`购买量向量`的内积，即为总消费。
	- **实例**:
		- 价格向量 $p = (\text{苹果单价}, \text{香蕉单价}, \text{牛奶单价}) = (2, 1, 3)$
			
		- 购买向量 $q = (\text{苹果数量}, \text{香蕉数量}, \text{牛奶数量}) = (5, 2, 4)$
			
		- 总消费 = $p \cdot q = (2 \times 5) + (1 \times 2) + (3 \times 4) = 24$ 元
	
- **做功**: 恒力F作用于物体，产生位移S，力F所做的功W就是F与S的内积。
	- 该例子印证了内积两种公式的等价性：
	- 按分量计算: $W = F_xS_x+F_yS_y$
	- 按投影计算: $W = F_{S}S=(|F|\cos\theta)|S|$
